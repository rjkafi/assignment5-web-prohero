Important FAQ

What is the DOM in JavaScript ?

The DOM (Document Object Model) represents an HTML document as a structured tree, allowing manipulation via JavaScript.

How do you select an element by its ID in JavaScript ?

Use document.getElementById('elementId') to select an element with a specific ID in the DOM.

How can you create a new HTML element using JavaScript?

Use document.createElement('tagName') to create a new HTML element, like div or p, in the DOM.

What does innerHTML do in JavaScript?

innerHTML sets or retrieves the HTML content inside an element, including child elements and text.

How do you remove an element from the DOM in JavaScript ?

Use element.remove() to remove an element from the DOM.
